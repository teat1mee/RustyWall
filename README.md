🦀 RustyWall

RustyWall — это современная, быстрая и безопасная CLI-утилита на Rust для управления обоями рабочего стола в среде GNOME. Забудьте о ручном скачивании картинок: теперь вы можете менять атмосферу вашего рабочего стола одной короткой командой в терминале.
🚀 Возможности

    🖼️ Локальный режим (-dir): Выбирает случайное изображение из указанной папки.

    🌐 Сетевой режим (-url): Скачивает изображение по прямой ссылке, автоматически извлекает имя файла и устанавливает его как фон.

    🛡️ Безопасность и надежность:

        Smart Parsing: Умное извлечение имен файлов из сложных URL (с параметрами запроса).

        User-Agent Spoofing: Обход блокировок со стороны серверов (представляется как браузер).

        Timeouts: Защита от зависаний при плохом интернет-соединении (таймаут 30 сек).

        Rust Safety: Полная обработка ошибок (никаких падений программы).

    🌒 Поддержка GNOME: Автоматически обновляет ключи как для светлой, так и для темной темы.

🛠 Установка
Для пользователей Rust (самый простой способ):

Вы можете установить rustywall прямо из GitHub-репозитория:
code Bash

cargo install --git https://github.com/teat1mee/RustyWall.git

После этого команда rustywall станет доступна в любом месте вашего терминала.
Сборка из исходников:

Если вы хотите собрать проект вручную:

    Клонируйте репозиторий:
    code Bash

    git clone https://github.com/teat1mee/RustyWall.git
    cd RustyWall

    Соберите и установите:
    code Bash

    cargo install --path .

📖 Использование

Запуск программы осуществляется через флаги.
Установить случайные обои из папки:
code Bash

rustywall -dir /home/user/Pictures/Wallpapers

Скачать и установить обои по ссылке:
code Bash

rustywall -url https://images.pexels.com/photos/15286/pexels-photo.jpg

Справка:
code Bash

rustywall -h

⚙️ Требования

    ОС: Linux (дистрибутивы с оболочкой GNOME).

    Среда: Утилита использует gsettings, которая предустановлена в большинстве современных дистрибутивов (Fedora, Ubuntu, Debian и др.).

📦 Стек технологий

    Rust (Edition 2021)

    Reqwest — для работы с сетью.

    Rand — для выбора случайных файлов.
